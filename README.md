# Бот-составитель расписания

Автоматизация составления драфта расписания учебных занятий для магистратуры «Прикладной искусственный интеллект».

## Описание

Чат-бот для Telegram, который общается с экспертами, составляет драфт расписания и согласовывает его.

### Минимальный функционал
- Первичный опрос эксперта (какие дни недели удобны)
- Сверка с уже имеющимся расписанием в табличном редакторе
- Подбор конкретных слотов (даты и время)
- Согласование получившегося расписания с экспертом в чате
- Внесение согласованных слотов в расписание или отправка продюсеру

## Структура проекта


## Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```
3. Активируйте виртуальное окружение:
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
5. Скопируйте `.env.example` в `.env` и заполните переменные окружения

## Запуск

```bash
python main.py
```

## Технологии

- Python 3.10+
- python-telegram-bot - для работы с Telegram API
- openpyxl/pandas - для работы с Excel
- pydantic - для валидации данных
